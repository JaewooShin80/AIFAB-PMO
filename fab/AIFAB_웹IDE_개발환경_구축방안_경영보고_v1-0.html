Overview

무엇을 만드나 시티즌 개발자용 표준 개발환경

목표. 과제 승인 즉시 팀별 개발환경이 자동 생성되고, 시티즌 개발자는 브라우저에서 SSO 로그인만 하면 Claude 하네스가 적용된 VS Code 환경에서 바로 개발을 시작한다.
1과제 승인승인 워크플로가 Terraform으로 팀 워크스페이스 자동 생성
2SSO 로그인사내 계정(Entra ID 연동)으로 브라우저 접속 — 설치·신청 없음
3IDE 즉시 시작VS Code 웹 환경 + Claude Code·AIFAB 하네스 선탑재 상태
4키리스 AI 연동Bedrock 경유 — API 키 배포 없이 권한(IAM)으로 자동 인증
