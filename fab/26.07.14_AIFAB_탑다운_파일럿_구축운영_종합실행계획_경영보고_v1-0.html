Readiness Review

셋업 완료 기준 8/28 킥오프 Go/No-Go

Track A — 시스템 구축 (AI 인프라팀)

  • 골든 패스 7대 템플릿 완비 (ECS·Lambda·RDS·비용태그·관측성·Secrets·AI 연동)
  • 폼 입력 → 레포 자동 생성 → 스테이징 배포 1시간 이내 (비개발자 드라이런 통과)
  • SAST·시크릿 검출 파이프라인 내장 (치명 취약점 0건 게이트 동작 확인)
  • 위험 등급(3-lane) 분류 기준·리뷰 절차 확정
  • 참여자 전원 계정·권한 발급, 무중단 배포·자동 롤백 검증
  • 네트워크 연결성·접속 권한·AI 호출 경로 실측 검증 (6개 기술 항목)

Track B — 프로그램 셋업 (AI Board)

  • 파일럿 예외 규정 승인 (기획안 주체 조항 보완)
  • 팀 2~3개 선발 확정, 부서장 3주 전담 서면 합의
  • KPI 6종·Go/No-Go 스코어카드 서명 동결
  • 전원 교육 이수 + 드라이런 배포 1회 성공
  • 멘토·기술상담 전담·챔피언 지정, 정기 기술상담(신청제)·전용 채널 개설
Go/No-Go 원칙. 전 항목 충족 시에만 킥오프 Go — 하나라도 미충족이면 1~2주 순연한다. 실패 확률이 높은 파일럿을 일정 때문에 강행하지 않는다.
